# ReqManager - IT 需求管理系统

IT 需求管理系统是一个帮助个人在整个项目生命周期中捕获、组织、跟踪和管理 IT 需求的 Web 平台。

## 功能特性

### 仪表盘
- 概览统计：需求总数、进行中、已完成、已逾期
- 最近需求列表
- 项目进度展示

### 需求管理
- 创建、编辑、删除需求
- 多维度筛选：状态、优先级、项目
- 全文搜索
- 需求详情查看
- **需求跟进** - 记录每次沟通的时间、地点、内容、结论

### 状态管理
- 自定义需求状态（名称、英文标识、颜色）
- 支持中英文切换显示
- 动态关联到需求列表和详情页

### 项目管理
- 项目卡片展示
- 需求进度可视化
- 快速跳转查看项目相关需求

### 用户管理
- 用户角色管理（管理员、经理、成员）
- 超级管理员专属账号

### 设置
- 状态配置管理（增删改查）

## 技术栈

- **前端**: Vue 3 + Vite + TypeScript
- **路由**: Vue Router 4
- **国际化**: Vue I18n
- **后端 API**: Node.js + Express + SQLite

## 安装运行

### 前置要求

- Node.js 18+
- npm 或 yarn

### 安装步骤

```bash
# 进入项目目录
cd req-manager

# 安装前端依赖
npm install

# 安装后端依赖
cd server
npm install
cd ..
```

### 启动服务

```bash
# 终端1：启动后端 API
cd server
npm start
# API 服务运行在 http://localhost:3001

# 终端2：启动前端
npm run dev
# 前端运行在 http://localhost:5173
```

访问 http://localhost:5173

## 使用指南

### 登录

使用超级管理员账号登录：

| 字段 | 值 |
|------|-----|
| 用户名 | Alice |
| 密码 | Aa135246@ |

### 创建需求

1. 进入「需求」页面
2. 点击右上角「新建需求」按钮
3. 填写表单并保存

### 需求跟进

1. 进入需求详情页
2. 点击「Add Follow-up」按钮
3. 填写跟进人、日期、地点、内容等信息
4. 支持编辑和删除已添加的跟进记录

### 管理状态

1. 点击侧边栏「Settings」
2. 在状态管理区域添加、编辑、删除状态
3. 新增/编辑需求时可以选择对应的状态

### 筛选需求

- 使用顶部筛选栏按状态、优先级、项目筛选
- 使用搜索框按标题搜索
- 点击「清除」恢复显示全部

### 查看项目需求

1. 进入「项目」页面
2. 点击任意项目的「查看需求」按钮
3. 将自动跳转到需求页面并筛选该项目
